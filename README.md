# Gestion Fanion

Projet de gestion scolaire minimal.

But: rendre le projet propre et professionnel.

Quick start
- Placez le dossier dans `C:\xampp\htdocs\gestion_fanion`
- Démarrez Apache/MySQL via XAMPP
- Importez la base depuis `Base de données.sql`
- Configurez `config.php` avec vos paramètres de base de données
- Accédez à `http://localhost/gestion_fanion/`

Notes
- Les fichiers `assets/` contiennent CSS, JS, images.
- Les PDFs sont générés via FPDF dans `includes/fpdf.php`.

Tâches recommandées après nettoyage
